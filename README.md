# Knowledge Map of Microbially-Mediated Dolomite Precipitation

## Project Overview

This project used Cytoscape to construct a knowledge network integrating biological, geochemical, and environmental factors associated with microbially-mediated dolomite precipitation.

The goal was to synthesize information from scientific literature and ontology-based databases into a visual network that highlights known relationships and identifies knowledge gaps related to the "dolomite problem."

## Scientific Question

How do microorganisms, environmental conditions, and geochemical processes interact to facilitate low-temperature dolomite precipitation in natural environments?

## Data Sources

Information was integrated from:

- Scientific literature
- NCBI
- UniProt
- Gene Ontology (GO)
- ENVO
- ChEBI

## Workflow

1. Review scientific literature
2. Identify biological, geochemical, and environmental entities
3. Construct nodes and edges tables
4. Import data into Cytoscape
5. Build and refine knowledge network
6. Visualize relationships and identify knowledge gaps

## Key Findings

- Multiple microbial groups contribute to dolomite precipitation through metabolic activity.
- Sulfate reduction and methanogenesis influence local geochemical conditions.
- Extracellular polymeric substances (EPS) appear to play an important role in mineral nucleation.
- Microbial communities alter pH, alkalinity, and sulfide concentrations that favor dolomite formation.
- Knowledge network construction highlighted remaining gaps in understanding the mechanisms underlying low-temperature dolomite precipitation.
